# Proiect de Practică: Minitopia - Proiectarea și Implementarea Platformei Digitale Premium

This repository contains the complete internship documentation and graduation-level practice project developed during my placement at the digital agency **Ideologiq (Advanced Ideas S.R.L.)**. The project focuses on the end-to-end design, business analysis, and architectural framework of a premium digital ecosystem for a custom children's playground named **Minitopia**.

## 📌 Project Overview & Structure
The project is structured into three main technical chapters, bridging business strategy with modern software engineering:

### 1. Corporate Placement & Agency Workflow (Chapter 1)
- Overview of **Ideologiq**, its digital portfolio, and legal framework.
- Analysis of organizational structures and departmental workflows (Project Management, UI/UX Design, Development, QA, and Marketing).
- Description of my role as an intern within the active agency pipelines.

### 2. Business and Product Analysis of Minitopia (Chapter 2)
- Definition of scope, mission, and unique value proposition in a premium niche market.
- Target audience segmentation and deep-dive Persona creation.
- Detailed functional requirements specifications, defining user interactions and core platform capabilities.

### 3. UI/UX Architecture and Frontend Engineering (Chapter 3)
- High-fidelity prototyping and interface layout concepts.
- Strict evaluation and implementation of **Jakob Nielsen’s 10 Usability Heuristics** for user interface design.
- Technical frontend framework formulation featuring **Next.js** implementation models, modern CSS Flexbox layout strategies, Custom Properties (variables), and dynamic font optimization pipelines.
- Rigid compliance architecture aligning user tracking, data flows, and registration forms with General Data Protection Regulation (**GDPR**) principles.
